Here are all 3D components in editable STEP and FreeCAD file formats as
well as STL for visualisation and 3MF for slicing.

We have used the following combinations of slicing and printing:

- Cura Ultimaker slicing for Prusa i3 MK3S+ and Anycubic i3 Mega.
- BambuLab Studio for BambuLab A1 mini.

All printed in generic PLA filament 1.75 mm.
